## How to contribute

1. Fork the repo and create a feature branch.
2. Make your change with tests.
3. Run the relevant test suite(s).
4. Open a PR with a clear description and rationale.

## Security

If you find a security issue, please report it privately (see `docs/SECURITY.md`).

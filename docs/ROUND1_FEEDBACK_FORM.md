```
Swarm-It ADK Round 1 Feedback Form
```

## Form Description
```
Please complete this form only after testing one of these Round 1 modules:

• Module A: Install Test
• Module C: Docs Clarity Review
• Module G: Video Walkthrough Feedback

This form is for feedback on the Swarm-It ADK, not for general app beta feedback.

Please give specific, honest, practical feedback based on real use.

Good feedback includes:
• the exact step where you got confused
• the exact error message
• the file, page, or link where the issue happened
• what would make the step easier

You may use AI for translation or clarification, but please report when you used it.
```
